코드작성, 디버깅, 배포..와 같은 일련의 과정을 
-------------------
[주피터 단축키]
 h :  설명
 셀 : 코드를 입력할 수 있는 공간 
 파란색 테두리 : 셀의 선택 
 연두색 테두리 : 셀에 코드 입력 모드 (셀의 클릭, 에터로 작동)
 esc : 선택모드로 빠져나오기 
 run 버튼, ctrl+ enter: 실행 
 B : 선택한 셀 아래에 새 셀 생성
 A : 선택한 셀 위에 새 셀 생성
 C : 선택한 셀 복사
 v : 선택한 셀 붙여넣기(아래에 붙여 넣어짐) 
 x : 선택한 셀 잘라내기 
dd : 선택한 셀 삭제
 z : 되돌리기
